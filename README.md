# World Sunlight
3D Globe with graphical display showing the cities with most/least sunlight

## Installation
`npm install`

## Running the server
`php -S 127.0.0.1:8002`

## Cron
`php cron.php`

## Live demo
https://marcelovani.github.io/world-sunlight
